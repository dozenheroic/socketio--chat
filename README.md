# Реализация чата на Socket.IO
## Установка

1. Клонирование репозитория:
   ```bash
   git clone https://github.com/whitespace-v/socketio-chat.git


Установка зависимостей:
npm install

Запуск сервера:
npm start

http://localhost:3000

## Как чат работает
Вход пользователя
При загрузке страницы пользователь вводит имя

Срабатывает событие add user, имя сохраняется

Общее число пользователей увеличивается

Показывается сообщение о входе и количество участников (login)

Отправка сообщений
Когда пользователь пишет сообщение и нажимает Enter:

Срабатывает событие new message

Сообщение и имя отправителя транслируются другим пользователям

Сообщение отображается в списке чата

Когда кто-то печатает
Если пользователь набирает текст:

Срабатывает событие typing

Отображается сообщение вида: username is typing

Когда пользователь прекращает печатать:

Срабатывает событие stop typing, надпись убирается

ОТКЛЮЧЕНИЕ ОТ ЧАТА
При закрытии вкладки или потере соединения:

Срабатывает событие disconnect

Пользователь удаляется из списка (numUsers)

Другие участники видят сообщение, что он вышел (user left)

Повторное подключение
При восстановлении соединения:
Срабатывает событие reconnect
Имя пользователя снова отправляется на сервер.

Если произошла ошибка:
Срабатывает reconnect_error, выводится сообщение об ошибке

## Отладка
Если нужно посмотреть, что происходит между клиентом и сервером:

DevTools → вкладка Network
Вкладка WS (WebSocket)

Там видны все события: login, new message, typing, user left и др.
